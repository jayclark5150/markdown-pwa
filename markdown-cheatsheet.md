# Markdown Cheatsheet

---

## Headings

```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

---

## Text Formatting

```
**bold**
*italic*
~~strikethrough~~
**_bold and italic_**
```

**bold** · *italic* · ~~strikethrough~~

---

## Lists

**Unordered**
```
- Item one
- Item two
  - Nested item
```

**Ordered**
```
1. First
2. Second
3. Third
```

**Task List**
```
- [x] Completed task
- [ ] Incomplete task
```

---

## Links & Images

```
[Link text](https://example.com)
![Alt text](image-url.png)
```

---

## Blockquotes

```
> This is a blockquote.
> It can span multiple lines.
```

---

## Code

**Inline code**
```
Use `backticks` for inline code.
```

**Code block**
````
```javascript
const greeting = "Hello, World!";
console.log(greeting);
```
````

Supported languages include: `javascript`, `python`, `bash`, `html`, `css`, `json`, `sql`, and many more.

---

## Tables

```
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell     | Cell     | Cell     |
| Cell     | Cell     | Cell     |
```

---

## Horizontal Rule

```
---
```

---

## Escaping Characters

Prefix any special character with `\` to display it literally:

```
\*not italic\*
\# not a heading
```

---

## Quick Reference

| Element       | Syntax                        |
|---------------|-------------------------------|
| Bold          | `**text**`                    |
| Italic        | `*text*`                      |
| Strikethrough | `~~text~~`                    |
| Inline code   | `` `code` ``                  |
| Link          | `[text](url)`                 |
| Image         | `![alt](url)`                 |
| Blockquote    | `> text`                      |
| Heading 1–4   | `#` through `####`            |
| Task item     | `- [ ]` / `- [x]`            |
| Horizontal rule | `---`                       |
