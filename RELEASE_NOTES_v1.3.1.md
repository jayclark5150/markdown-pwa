# v1.3.1 Release Notes

## What's New

**Line Numbers Toggle**
A new button in the toolbar lets you show or hide line numbers in the editor. Click the line numbers icon (in the Find group) to toggle. State resets on reload.

**Copy Button Always Visible**
The copy button on code blocks is now always visible rather than appearing only on hover, making it easier to discover and use on touch devices.

**Toolbar Hide Button**
A tab on the left edge of the toolbar lets you collapse the entire toolbar to maximize writing space.

## Fixes

- Service worker cache bumped to v4 to ensure all users load the latest files
